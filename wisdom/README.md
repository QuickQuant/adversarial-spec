# Wisdom Directory

This directory stores cross-session learnings that accumulate over time.

## Structure

```
wisdom/
├── general/              # Role-based wisdom (applies to all projects)
│   ├── ceo-wisdom.md
│   └── manager-wisdom.md
└── adversarial-spec/     # Project-specific learnings
    └── ceo-learnings.md
```

## Guidelines

- Wisdom files grow organically through SmartCompact
- Content should be patterns, not task-specific details
- General wisdom may be synced across projects
- Project wisdom stays local
